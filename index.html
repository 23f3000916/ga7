Financial Modeling: CAGR

We use the Compound Annual Growth Rate (CAGR) to measure long-term growth.

The formula is:

$$ CAGR = \left( \frac{EV}{BV} \right)^{\frac{1}{n}} - 1 $$

Where:

  • $EV$ = Ending Value
  • $BV$ = Beginning Value
  • $n$ = Number of years

Risk Analysis Script

Python script for calculating Value at Risk (VaR).


import numpy as np

# Calculate 95% VaR for a portfolio
def calculate_var(returns, confidence_level=0.95):
    sorted_returns = np.sort(returns)
    index = int((1 - confidence_level) * len(sorted_returns))
    var = sorted_returns[index]
    return abs(var)

# Example usage:
portfolio_returns = np.random.normal(-0.001, 0.01, 1000)
value_at_risk = calculate_var(portfolio_returns)
print(f"95% VaR: {value_at_risk:.2%}")
